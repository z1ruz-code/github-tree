# 🌳 GitHub Tree - профиль как дерево
Превратите любой публичный профиль GitHub в живое, растущее дерево.
Один репозиторий = одна ветка. Много листьев, физика ветра, анимация роста и интерактивность.

**Живое демо**: https://z1ruz-code.github.io/github-tree

---

## ✨ Особенности

- 🌿 1 репо → 1 ветка
Количество и «густота» дерева зависят от числа репозиториев пользователя. Чем больше проектов, тем пышнее крона.

- 🍃 Настоящая физическая симуляция
Ветки и листья колышутся под порывами ветра. Реализована физика с пружинами, демпфированием и динамической силой ветра.

- 🌱 Анимация роста
Дерево не появляется мгновенно - оно вырастает из земли за несколько секунд, ветки разворачиваются плавно и естественно.

- 💬 Подсказки при наведении
Наведите курсор на любую ветку, и появится карточка с названием репозитория, описанием, языком и звёздами.

- 🔗 Переход по клику
Кликните по ветке - откроется страница соответствующего репозитория на GitHub в новой вкладке.

- 🎨 Приятный минималистичный дизайн
Стеклянная панель управления, мягкий градиент неба и земли, падающие листья, плавающие частицы, виньетка.

- 🚀 Работает полностью на клиенте
Никаких серверов, баз данных или ключей API. Просто откройте файл в браузере.

## 🛠️ Технологии

- **HTML5 Canvas** - вся графика и анимация

- **CSS3** - стилизация интерфейса, эффекты размытия, адаптивность

- **JavaScript (ES6)** - физический движок, запросы к GitHub API, взаимодействие

- **[GitHub REST API](https://docs.github.com/rest)** - получение публичных данных о репозиториях (без токена)

Никаких фреймворков или внешних библиотек. Всё в одном файле.

---

## ⚙️ Как это работает

1.  Вы вводите никнейм → приложение делает запрос к api.github.com/users/{username}/repos

2.  Полученные репозитории преобразуются в ветви дерева: каждый репо становится основной веткой, от которой отходят более мелкие, а на них — десятки листьев.

3.  Физика обновляется 60 раз в секунду: ветер, инерция, упругость веток и колебания листьев.

4.  При движении мыши проверяется расстояние до каждой ветки с репо. Если курсор близко - показывается тултип. При клике — открывается html_url.

Лимиты GitHub API без авторизации: 60 запросов в час с одного IP. При превышении появится ошибка.
